# Cross-Corpus Enriched Cross-Reference Index

This folder holds **cross-corpus bridge documents** produced by the `cross-corpus-enrichment-analysis` skill.

Each document links two corpora — one **dominant**, one **secondary** — and maps:

- transfer chains (secondary mechanism → dominant family → project consequence)
- bridge sources and bridge concepts
- cross-corpus tensions
- joint trend lines
- consequences for the project's goals

## Naming convention

`cross_reference_index_<corpus1_short>_<corpus2_short>_enriched_<N>_time.md`

Passes are incremental: each new pass for a corpus pair is additive, not a rewrite.

This folder starts empty; bridge documents accumulate as corpora mature.
